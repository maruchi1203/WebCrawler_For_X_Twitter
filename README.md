# 트위터 관리자인 친구한테 줄 X(Twitter) 전용 웹 크롤러

## 기능
1. 로그인 정보를 등록하여 X에 자동 로그인
2. X의 특정 계정이 일정 기간동안 올린 모든 글의 정보를 HTML 태그와 Text 기준으로 크롤링
3. 엑셀 파일(전용 템플릿 필요)을 지정하여 글의 정보(내용, 댓글 수, 좋아요 수 등)를 시간 역순으로 기록

## 사용 기술 스택
- Selenium : 웹 크롤링
- PyQt5 : GUI
- openpyxl : 엑셀 활용

## 참고
- 과거에 개발했기 때문에 현재는 작동하지 않을 가능성이 높습니다.
- 위 프로그램은 공개된 데이터의 제한적인 정보 수집을 목적으로 하고 있습니다.
- 로그인 정보를 입력하지 않는다면 사용할 수 없습니다.
